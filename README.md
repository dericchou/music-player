# music-player

> 本项目采用Canvas、JavaScript、CSS3、SCSS、Gulp以及面向对象编程实现一个基于H5的音乐播放器,本音乐播放器支持歌词同步、歌曲切换、播放、暂停、上一曲、下一曲、背景更换、播放进度条、调节音量、鼠标拖拽等功能


技术栈
---

- HTML5
- JavaScript
- jQuery
- SCSS

Usage
---

```
npm install
npm run dev
npm run build
```
